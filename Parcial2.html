border-collapse: collapse

Para colapsar los bordes y las celdas queden unidas el cual se aplica a las tablas

border: 1px solid black

Para asignar un borde de color negro, con una forma de línea continua y un tamaño de 1px de grosor y se está aplicando a los td

border: 0px

Se crea la clase anterior para ser aplicada en los bordes de algunos td

style="text-align: center

Para centrar el texto contenido en este caso se utilizó para centrar la información del estudiante

style="padding: 0%

En el campo que corresponde a Lorem, se elimina un espacio generado a nivel interno de td

style="background-color: grey

Se utiliza en el pie de página donde se ingresa la información de la universidad, esto con el fin que sea visible el logo

style="margin-bottom: 0px

Reduce a un tamaño de 0px la separación inferior del último párrafo

text-align: justify

Justifica el texto en ambos extremos

text-align: left

Alinea el texto a la izquierda

text-align: right

Alinea el texto a la derecha

border-collapse: separate

Separa las celdas según el tamaño deseado

font-size

determina el tamaño de la fuente

font-family

indica el tipo de fuente a mostrar, esta opción carga al menos tres tipos de fuente por si alguna de las principales no está disponible en el ordenador en que se abre la página web

font-style:oblique

pone en cursiva el texto

border-left

Indica el grosor del borde izquierdo

border-color

Indica el color que será asignado al contorno de la celda

border-style

cambia el tipo de línea o incluso puede usarse para que el borde no sea visible

style="width :

Indica el ancho de un objeto en la página web como lo puede ser una imagen

style="height”:

Indica el alto de un objeto en la página web como lo puede ser una celda

margin:

Genera un espacio entre etiquetas ya sea en la parte superior o a los lados

padding:

permite asignar o retirar un espacio entre objetos a nivel interno de una celda
