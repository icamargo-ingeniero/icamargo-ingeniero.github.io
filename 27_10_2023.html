    En el contenido actual de esta página se están usando algunos comandos de JavaScript que son :

    El siguiente botón este configurado con el evento onclick de tal manera que cada vez que se dé un clic sobre el ejecutara el java script que tiene relacionado

    El primer uso de JavaScript como se logra percibir al ingreso de la página es una bienvenida con un mensaje emergente, dentro de la codificación este mensaje se alerta se conoce como

    window.alert(" XXXXXXXXX ");

    El segundo y tercero no son visibles para el usuario, ya que estas son dos variables que interactuaran con un botón, las variables son colores e índiceColor, la primera es un vector en el cual se declara una serie de colores y la variable indiceColor inicializa el objeto en 0 para su uso

    var colores = ["black", "red", "blue", "green", "#9fedf3", "yellow", "white"];

    var indiceColor = 0;

    El cuarto uso que tenemos es el de una función que se llama fondo2

    Dentro de esta función se esta indicando que en el documento (hace referencia al contenido del .html), debe buscar un elemento el cual pertenece a la familia de los Id cuyo nombre es fondo; cuando lo haya encontrado va asignarle un nuevo valor al fondo, y va ser el valor que trae la variable indiceColor

    document.getElementById("fondo").style.backgroundColor = colores[indiceColor];

    y para finalizar la función tenemos el un reinicio de la variable, que en el momento que llegue a su máximo, esta será establecida nuevamente en 0 con el uso de % y de esta manera otra vez iniciar desde el primer color del arreglo

    indiceColor = (indiceColor + 1) % colores.length;
